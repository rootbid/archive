# Sentinel Threat Intelligence: 2026-03-08

## Executive Summary
This brief details four critical vulnerabilities actively impacting infrastructure and AI systems as of March 8, 2026. High-priority attention is required for Cisco SD-WAN deployments and Java-based authentication flows using `pac4j`. A new critical RCE in OpenClaw highlights the growing attack surface of autonomous AI agents.

## 1. Cisco Catalyst SD-WAN Authentication Bypass (CVE-2026-20127)
**Severity:** Critical (CVSS 10.0) | **Status:** Actively Exploited

*   **Description:** A maximum-severity authentication bypass vulnerability in Cisco Catalyst SD-WAN Controller and Manager allows unauthenticated remote attackers to gain administrative access.
*   **Attack Vector:** Exploitation targets the NETCONF service, typically exposed on TCP port 830.
*   **Impact:** Full system compromise, allowing attackers to reconfigure network routing and intercept traffic across the SD-WAN fabric.
*   **Mitigation:**
    *   **Patch:** Apply emergency patches immediately (releases 20.12.x and later).
    *   **Workaround:** Restrict access to TCP/830 via ACLs to trusted management IPs only.

## 2. pac4j-jwt Authentication Bypass (CVE-2026-29000)
**Severity:** Critical (CVSS 10.0) | **Status:** PoC Available

*   **Description:** A flaw in the `pac4j-jwt` library allows attackers to bypass authentication by forging a JWE-wrapped PlainJWT. This effects versions prior to 4.5.9, 5.7.9, and 6.3.3.
*   **Attack Vector:** An attacker with knowledge of the server's RSA public key (often exposed) can craft a token that the server accepts as valid for *any* user, including administrators.
*   **Impact:** Total authentication bypass leading to privilege escalation and potential data exfiltration.
*   **Mitigation:**
    *   **Upgrade:** Update `pac4j-jwt` to versions 4.5.9, 5.7.9, 6.3.3 or later.
    *   **Detection:** Audit logs for unexpected administrative logins or JWTs with unusual signing algorithms.

## 3. OpenClaw RCE & Token Theft (CVE-2026-25253)
**Severity:** High (CVSS 8.8) | **Status:** Disclosed

*   **Description:** A remote code execution vulnerability in the OpenClaw AI assistant framework allows attackers to inject commands and steal authentication tokens.
*   **Attack Vector:** Exploitation occurs via prompt injection or malicious input handling within the agent's runtime environment.
*   **Impact:** Compromise of the host machine running the agent, theft of API keys (LLM providers, tools), and lateral movement.
*   **Mitigation:**
    *   **Patch:** Update OpenClaw core to the latest stable release immediately.
    *   **Hardening:** Run agents in sandboxed environments (Docker/Firecracker) with strictly limited network egress.

## 4. Android System Remote Code Execution (CVE-2026-0006)
**Severity:** Critical | **Status:** Patch Released (2026-03-01)

*   **Description:** A zero-click RCE vulnerability in the core Android System component.
*   **Attack Vector:** Remote exploitation without user interaction, likely via specially crafted wireless packets or media files.
*   **Impact:** Full device compromise.
*   **Mitigation:**
    *   **Update:** Apply the 2026-03-01 security patch level immediately.
