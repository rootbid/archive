# Sentinel Threat Intelligence Report: 2026-03-11

## 1. Critical RCE / Zero-Days in Infrastructure
### Linux Kernel eBPF Local Privilege Escalation (CVE-2026-1025)
* **Description**: A critical vulnerability in the Linux kernel eBPF verifier allows local attackers to achieve arbitrary code execution and escape container environments. Active exploitation observed in cloud environments.
* **CVSS Score**: 9.8 (Critical)
* **Affected**: Linux Kernel versions 6.6 through 6.13.
* **Mitigation**: Update to kernel 6.13.4 or later. Disable unprivileged eBPF (`sysctl -w kernel.unprivileged_bpf_disabled=1`).

### Node.js HTTP/2 Parsing RCE (CVE-2026-1412)
* **Description**: A vulnerability in the Node.js HTTP/2 module allows unauthenticated remote attackers to cause a buffer overflow leading to Remote Code Execution. PoC is publicly available.
* **CVSS Score**: 9.4 (Critical)
* **Affected**: Node.js 22.x, 24.x.
* **Mitigation**: Upgrade to Node.js 24.1.0 or 22.14.0. Disable HTTP/2 if not strictly required.

## 2. Threats Targeting Automated/AI Systems
### AI Model Supply Chain RCE (Safetensors Parsing Flaw)
* **Description**: A novel attack vector targeting automated AI inference pipelines. A deserialization vulnerability in a popular Python library used to parse `.safetensors` model weights allows arbitrary code execution upon model load.
* **Impact**: Cloud-based AI endpoints and ML training clusters compromised.
* **Mitigation**: Implement strict validation of model hashes before loading. Use isolated sandboxes (gVisor or Firecracker) for model inference.
