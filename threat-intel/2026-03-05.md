# Sentinel Threat Intelligence Brief: March 5th, 2026

## 1. Critical Infrastructure RCE: Ivanti EPMM Zero-Days
**Severity:** Critical (CVSS 9.8) | **Status:** Active Exploitation
**CVEs:** CVE-2026-1281, CVE-2026-1340

Two zero-day vulnerabilities in Ivanti Endpoint Manager Mobile (EPMM) are being actively exploited in the wild. These flaws allow unauthenticated attackers to execute arbitrary code on the appliance, granting full control over mobile device management infrastructure.

*   **Attack Vector:** Remote, Unauthenticated.
*   **Impact:** Full system compromise, lateral movement potential via MDM managed devices.
*   **Mitigation:** 
    *   Apply Ivanti patches immediately (released late Jan/Feb 2026).
    *   Isolate EPMM appliances from the public internet if patching is delayed.
    *   Audit logs for suspicious administrative access or file creation in web directories.

## 2. OpenSSL Remote Code Execution (CVE-2025-15467)
**Severity:** Critical | **Status:** PoC Available / Patch Released

A critical vulnerability in OpenSSL (versions 3.0, 3.3, 3.4, 3.5, 3.6) allows for potential remote code execution. The flaw exists in the CMS AuthEnvelopedData structure where an oversized Initialization Vector (IV) is copied into a fixed-size stack buffer without length verification.

*   **Attack Vector:** Remote, requires processing maliciously crafted CMS data.
*   **Impact:** Stack buffer overflow leading to crash (DoS) or RCE.
*   **Mitigation:** 
    *   Upgrade to the latest OpenSSL 3.x patch releases immediately.
    *   Monitor for crashes in services processing external CMS/PKCS#7 data.

## 3. AI & Cloud Infrastructure Threats
**Microsoft Copilot Data Leak (Patched Feb 19, 2026)**
*   **Threat:** A logic flaw allowed Copilot to index and surface data from "Sent Items" and "Drafts" in Outlook, bypassing configured confidential labels.
*   **Impact:** Unintentional exposure of sensitive internal communications to AI users within the tenant.
*   **Mitigation:** Ensure Microsoft 365 tenants are updated; review data governance policies for AI indexing scope.

**"PromptSpy" AI-Targeting Malware**
*   **Threat:** New Android malware strain "PromptSpy" actively abuses on-device AI capabilities (specifically Gemini integration points) to automate persistence mechanisms and evade behavioral detection.
*   **Impact:** Advanced persistence on mobile endpoints; demonstrates evolution of malware leveraging local AI APIs.
*   **Mitigation:** Enforce strict application allow-listing on corporate mobile devices; monitor for unusual API usage patterns in EDR telemetry.
