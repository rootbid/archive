# Sentinel Threat Intelligence Brief: 2026-02-27

**Status:** RED // CRITICAL  
**Audience:** Infrastructure/AppSec/SOC  
**Date:** Friday, February 27th, 2026

## Executive Summary
Multiple zero-day vulnerabilities are currently under active exploitation in the wild, targeting widespread enterprise infrastructure (Cisco, Ivanti) and client-side execution environments (Chrome/Chromium). Additionally, a new class of "AI Recommendation Poisoning" attacks has been observed targeting LLM-integrated summarization tools.

---

## 1. Critical Infrastructure RCEs (Active Exploitation)

### **Cisco SD-WAN vManage: Unauthenticated RCE (Zero-Day)**
*   **CVE:** `CVE-2026-20127`
*   **Severity:** **Critical (CVSS 10.0)**
*   **Impact:** Complete system compromise. Allows unauthenticated attackers to execute arbitrary code as `root` via the cluster management APIs.
*   **Status:** **Active exploitation confirmed.** CISA has added this to the KEV catalog. Attackers have been exploiting this since late 2023, but disclosure occurred in Feb 2026.
*   **Mitigation:**
    *   **Patch:** Upgrade to Cisco SD-WAN vManage releases `20.12.x` or later immediately.
    *   **Workaround:** Restrict access to the cluster management API to trusted internal IPs only via ACLs.

### **Ivanti EPMM (MobileIron): Pre-Auth Command Injection**
*   **CVE:** `CVE-2026-1281`
*   **Severity:** **Critical (CVSS 9.8)**
*   **Impact:** Remote Code Execution (RCE). An unauthenticated attacker can execute arbitrary commands on the appliance OS via malicious HTTP requests targeting outdated Bash scripts used by Apache `RewriteMap`.
*   **Status:** **Widely exploited.** 
*   **Mitigation:**
    *   **Patch:** Apply immediate updates for Ivanti EPMM versions `12.5.0.0` through `12.7.0.0`.
    *   **Defense:** Monitor logs for abnormal HTTP requests containing Bash shell metacharacters.

## 2. Client-Side & Runtime Threats

### **Google Chrome/Chromium: Zero-Day RCE**
*   **CVE:** `CVE-2026-2441`
*   **Severity:** **High (Zero-Day)**
*   **Impact:** Arbitrary Code Execution. A Use-After-Free vulnerability in the Blink CSS engine allows attackers to execute code via malicious web content (specifically targeting `@font-feature-values`). Sandbox escape requires chaining with a kernel exploit.
*   **Status:** **Active exploitation.** Emergency patches released ~Feb 13, 2026.
*   **Mitigation:**
    *   **Update:** Update Chrome/Chromium-based browsers (Edge, Brave, Electron apps) to the latest version immediately.
    *   **Hunting:** Look for crashes or unexpected behavior in the renderer process associated with font rendering.

### **Linux Kernel: High-Severity Remote Vectors**
*   **Advisories:** `RHSA-2026:3388`, `RHSA-2026:3375` (Red Hat / Upstream Linux)
*   **Severity:** **High (CVSS 9.8)**
*   **Impact:** Remote Attack Vectors identified in kernel subsystems (specifics vary by distribution, often involving network-facing drivers or subsystems like SMB/CIFS).
*   **Mitigation:** Prioritize kernel patching for all internet-facing Linux systems.

## 3. Emerging AI & Supply Chain Threats

### **AI Recommendation Poisoning ("Slopsquatting" & Content Injection)**
*   **Threat:** Attackers are embedding hidden instructions in web content (e.g., hidden HTML comments, white-on-white text) specifically designed to manipulate "Summarize with AI" features in browsers and enterprise tools.
*   **Impact:** Can cause AI agents to hallucinate non-existent packages (leading to supply chain attacks via "slopsquatting") or inject malicious links into summaries.
*   **Observation:** Microsoft researchers identified a campaign in mid-February 2026 using this technique for promotional and malicious redirect purposes.

### **Soliton FileZen: OS Command Injection**
*   **CVE:** `CVE-2026-25108`
*   **Severity:** **High (CVSS 8.7)**
*   **Impact:** Authenticated OS Command Injection in the file transfer appliance.
*   **Status:** **Active Exploitation confirmed by CISA.**
