# Sentinel Threat Intelligence Brief: 2026-02-27

**Status:** CRITICAL
**Scope:** Infrastructure, Remote Access, Browser
**Time:** 2026-02-27 13:51 UTC

## Executive Summary
Multiple high-severity zero-day vulnerabilities are under active exploitation in the wild. Immediate patching is required for Cisco SD-WAN environments and BeyondTrust remote access appliances. A new Chrome zero-day affecting CSS parsing is being used to compromise clients.

## 1. Cisco Catalyst SD-WAN Zero-Day (CVE-2026-20127)
**Severity:** Critical (CVSS 10.0)
**Type:** Authentication Bypass / RCE
**Status:** Active Exploitation (Confirmed since 2023)

A catastrophic vulnerability in Cisco Catalyst SD-WAN Controller (vSmart) and Manager (vManage) allows unauthenticated remote attackers to bypass authentication and gain root-level privileges.

*   **Impact:** Full system compromise, ability to manipulate network traffic, intercept data, and inject malicious configurations across the SD-WAN fabric.
*   **IoCs:** Check logs for unauthorized administrative logins or configuration changes originating from unknown IPs.
*   **Mitigation:** IMMEDIATE update to fixed releases (20.12.x or later). Isolate management interfaces.

## 2. BeyondTrust RS/PRA RCE (CVE-2026-1731)
**Severity:** Critical (CVSS 9.9)
**Type:** OS Command Injection
**Status:** Active Exploitation (VShell, SparkRAT)

An unauthenticated attacker can inject OS commands via specially crafted HTTP requests to the `thin-scc-wrapper` component in Remote Support (RS) and Privileged Remote Access (PRA) appliances.

*   **Impact:** Remote Code Execution (RCE) as root. Attackers are deploying VShell and SparkRAT for persistence.
*   **IoCs:**
    *   Processes: `spark`, `vshell`
    *   Network: Outbound connections to unknown IPs on high ports.
    *   Logs: Abnormal HTTP requests to `/api/*` endpoints.
*   **Mitigation:** Apply patches immediately. Ensure appliances are not exposed to the public internet if possible.

## 3. Google Chrome CSS Zero-Day (CVE-2026-2441)
**Severity:** High
**Type:** Use-After-Free (UAF) in CSS Parser
**Status:** Active Exploitation

A zero-day vulnerability in Chrome's CSS parsing engine allows attackers to corrupt heap memory via malicious CSS, leading to RCE without relying on JavaScript.

*   **Impact:** Client-side RCE, sandbox escape potential when chained.
*   **Vectors:** Malicious web pages, compromised legitimate sites.
*   **Mitigation:** Update Chrome/Chromium to the latest version immediately.

## 4. Ivanti EPMM Zero-Days (CVE-2026-1281 & CVE-2026-1340)
**Severity:** Critical (CVSS 9.8)
**Type:** RCE & Authentication Bypass
**Status:** Active Exploitation

Two vulnerabilities in Ivanti Endpoint Manager Mobile (EPMM) allow unauthenticated attackers to execute arbitrary code and take over MDM servers.

*   **Impact:** Full control over mobile device fleets, data theft, persistence via web shells.
*   **Mitigation:** Apply hotfixes provided by Ivanti.
