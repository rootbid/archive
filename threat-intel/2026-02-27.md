# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This brief highlights critical security threats identified in late February 2026, focusing on infrastructure, AI automation platforms, and active zero-day exploitation. Immediate action is required for systems running **n8n**, **Google Chrome/Chromium**, and specific **Linux Kernel** configurations.

## 1. Critical Infrastructure & Automation
### n8n Workflow Automation Platform (RCE & Sandbox Escape)
**Severity:** Critical (CVSS 9.9)
**Vulnerabilities:**
*   **CVE-2026-25049:** Remote Code Execution (RCE) via arbitrary command injection.
*   **CVE-2026-1470:** Sandbox escape in the JavaScript expression engine, allowing attackers to break out of the execution environment.
*   **GHSA-mqpr-49jj-32rc:** Webhook forgery permitting unsigned POST requests to trigger workflows with arbitrary data.
**Impact:** Unauthenticated attackers can achieve full server compromise, stealing stored credentials (API keys, OAuth tokens, database passwords) and pivoting into connected cloud environments.
**Mitigation:** Update n8n instances immediately to the latest patched version (v1.x latest / v2.x). Verify webhook configurations to enforce signature validation.

## 2. Active Zero-Day Exploitation
### Google Chrome / Chromium (Renderer RCE)
**Severity:** Critical
**Vulnerability:** **CVE-2026-2441**
**Status:** **Active exploitation in the wild.**
**Details:** A zero-day vulnerability in the Chrome renderer process. While the sandbox initially limits impact, attackers are chaining this with sandbox escapes to achieve full system compromise. Public Proof-of-Concept (PoC) code demonstrates three distinct trigger paths.
**Impact:** Arbitrary code execution on victim machines visiting malicious sites.
**Mitigation:** Update Google Chrome and all Chromium-based browsers (Edge, Brave, etc.) immediately.

## 3. Linux Kernel & Infrastructure
### Privilege Escalation & Memory Corruption
**Vulnerabilities:**
*   **CVE-2026-23074:** Use-after-free in the `teql` queueing discipline leading to privilege escalation.
*   **CVE-2025-68800:** `page_pool` stats vulnerability.
*   **CVE-2025-40064:** Out-of-bounds write in `fbdev`.
**Impact:** Local attackers can escalate privileges to root or cause denial of service (DoS).
**Mitigation:** Apply vendor kernel patches immediately.
*   **Ubuntu:** Updates released for 25.10, 24.04 LTS, 22.04 LTS (USN-8059-6).
*   **AlmaLinux/RHEL:** Updates available for `kernel` and `kernel-rt` packages.
