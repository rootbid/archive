# Sentinel Threat Intelligence: 2026-02-27

## Executive Summary
This report details critical security threats actively exploited in the wild or possessing immediate high-impact potential as of February 27, 2026. Focus areas include Zero-Day RCEs in browser infrastructure, critical flaws in MDM/SD-WAN systems, and targeted attacks against AI development environments.

## 1. Critical Infrastructure & Zero-Day Exploits

### CVE-2026-2441: Chrome/Chromium Zero-Day RCE
*   **Severity:** **CRITICAL** (Active Exploitation Confirmed)
*   **Impact:** Remote Code Execution (RCE) via malicious web content.
*   **Target:** Google Chrome and Chromium-based browsers (Edge, Brave, Electron apps).
*   **Status:** A patch has been released. Attackers are actively using this zero-day in the wild to compromise users visiting malicious sites.
*   **Mitigation:**
    *   Update Chrome/Chromium to the latest version immediately.
    *   Restart browser instances to apply updates.
    *   Monitor endpoint logs for unusual child processes spawned by browser executables.

### CVE-2026-20127: Cisco SD-WAN vManage Command Injection
*   **Severity:** **CRITICAL (CVSS 10.0)**
*   **Impact:** Unauthenticated RCE leading to root access.
*   **Target:** Cisco SD-WAN vManage.
*   **Status:** Actively exploited in the wild (exploitation observed as far back as late 2023). Added to CISA KEV.
*   **Mitigation:**
    *   Apply emergency patches provided by Cisco immediately.
    *   Restrict access to the vManage management interface to trusted internal networks only.

### CVE-2026-1281 & CVE-2026-1340: Ivanti EPMM Zero-Days
*   **Severity:** **CRITICAL (CVSS 9.8)**
*   **Impact:** Unauthenticated Remote Code Execution.
*   **Target:** Ivanti Endpoint Manager Mobile (EPMM), formerly MobileIron Core.
*   **Status:** Active exploitation confirmed. Attackers can bypass authentication and execute arbitrary commands on the appliance.
*   **Mitigation:**
    *   Apply the latest security patches from Ivanti.
    *   Isolate affected appliances from the public internet if patching is not immediately possible.
    *   Audit logs for suspicious administrative access or configuration changes.

## 2. AI & Automation Systems

### Claude Code RCE (CVE-2026-21852 / CVE-2025-59536)
*   **Severity:** **HIGH**
*   **Impact:** Remote Code Execution and API Token Exfiltration.
*   **Target:** Developers using Anthropic's "Claude Code" tool and AI-assisted development environments.
*   **Vector:** Malicious project configurations. When a developer opens a compromised project or repository with Claude Code, hidden directives can execute arbitrary system commands or exfiltrate API keys.
*   **Mitigation:**
    *   Update Claude Code tools to the latest patched version.
    *   Exercise extreme caution when cloning untrusted repositories or opening project files from unknown sources.
    *   Run AI coding tools in isolated/sandboxed environments (e.g., containers, VMs) to limit the blast radius of potential RCE.
