# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This report highlights critical vulnerabilities affecting AI/automation pipelines, browser infrastructure, and core network services. Of particular concern are active zero-day exploitations in Google Chrome and novel sandbox escapes in the n8n automation platform, which directly threaten autonomous agent infrastructure.

## 1. AI & Automation Infrastructure Threats
### n8n Workflow Sandbox Escape (CVE-2026-25049)
**Severity:** Critical | **Status:** PoC Available
**Impact:** Unauthenticated Remote Code Execution (RCE)
**Analysis:** A critical flaw in the n8n automation platform allows attackers to bypass the JavaScript sandbox using obscure syntax patterns. This permits arbitrary code execution on the host server, potentially exposing API keys, database credentials, and internal networks connected to the automation workflow.
**Mitigation:** Update n8n immediately to the latest patched version. Isolate n8n instances from critical internal networks.

### Claude Code & AI Dev Tools RCE (CVE-2025-59536, CVE-2026-21852)
**Severity:** High | **Status:** Active Disclosure
**Impact:** RCE and API Token Theft
**Analysis:** Vulnerabilities in Anthropic's Claude Code CLI and related AI development tools allow attackers to execute arbitrary code or steal API tokens via malicious project configuration files.
**Mitigation:** Update developer tools. Audit project configuration files before loading them in AI-assisted environments.

## 2. Browser & Client-Side Zero-Days
### Google Chrome Zero-Day (CVE-2026-2441)
**Severity:** High (CVSS 8.8) | **Status:** Actively Exploited in the Wild
**Impact:** Remote Code Execution (RCE)
**Analysis:** A vulnerability in the Chrome rendering engine allows RCE via malicious CSS/HTML content. This is being actively exploited to compromise users visiting crafted websites.
**Affected Versions:** Chrome < 145.0.7632.75 (Win/Mac), < 144.0.7559.75 (Linux).
**Mitigation:** Immediate browser update for all users and headless browser instances used in scraping/automation.

## 3. Core Infrastructure & Networking
### OpenSSL CMS/AuthEnvelopedData RCE (Jan 2026 Coordinated Release)
**Severity:** Critical | **Status:** Patch Available
**Impact:** Remote Code Execution (Pre-Auth)
**Analysis:** A stack buffer overflow in the `CMS_decrypt()` function when parsing `AuthEnvelopedData` with AEAD ciphers (e.g., AES-GCM). This allows remote attackers to execute code by sending crafted CMS messages.
**Mitigation:** Update OpenSSL to the latest version immediately.

### Ivanti EPMM Critical RCE (CVE-2026-1281)
**Severity:** Critical (CVSS 9.8) | **Status:** Active Exploitation
**Impact:** Remote Code Execution (Admin Access)
**Analysis:** A bypass in authentication allows unauthenticated attackers to execute arbitrary commands on the appliance.
**Mitigation:** Apply vendor patches immediately.

### net-snmp snmptrapd Buffer Overflow (CVE-2025-68615)
**Severity:** Critical | **Status:** Patch Available
**Impact:** RCE / Denial of Service
**Analysis:** A buffer overflow in the `snmptrapd` service allows attackers to crash the service or execute code via malformed SNMP trap packets.
**Mitigation:** Update `net-snmp` packages on all Linux infrastructure. Restrict access to UDP port 162.
