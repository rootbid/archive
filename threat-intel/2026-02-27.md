# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This brief highlights critical security threats active as of late February 2026, focusing on infrastructure zero-days and emerging attacks against AI systems.

**Top Priority:** Immediate patching of Chromium-based environments (browsers, Electron apps, headless CI/CD) due to active in-the-wild exploitation of a new RCE zero-day.

## 1. Critical Infrastructure & RCE

### CVE-2026-2441: Chrome/Chromium Blink Engine Use-After-Free
*   **Severity:** Critical (CVSS 8.8)
*   **Status:** **Active In-The-Wild Exploitation** / PoC Available
*   **Affected:** Google Chrome < 146.0.7500.12, Chromium, Edge, Electron-based apps, Headless Chrome in CI/CD pipelines.
*   **Technical Detail:** A Use-After-Free (UAF) vulnerability in the Blink rendering engine's CSS handling allows remote attackers to execute arbitrary code via a crafted HTML page.
*   **Impact:** Full system compromise of the user's machine or container escape in headless environments.
*   **Mitigation:** Update to version 146.0.7500.12+ immediately. Restart all browser instances.

### CVE-2026-25108: Soliton FileZen Privilege Escalation
*   **Severity:** High (CVSS 8.7)
*   **Status:** **Active Exploitation** (Confirmed by CISA KEV)
*   **Affected:** FileZen versions 4.2.1–4.2.8 and 5.0.0–5.0.10.
*   **Technical Detail:** Improper neutralization of special elements allows authenticated attackers (or those chaining with auth bypass) to escalate privileges to root or execute commands.
*   **Mitigation:** Apply vendor patches immediately. Isolate appliances from the public internet if patching is delayed.

## 2. AI & Automation Systems Targeting

### OpenClaw SSRF & Malicious Skills Campaign
*   **CVE:** CVE-2026-27488
*   **Status:** Fixed in v2026.2.19 / Active Malicious Skills Campaign
*   **Technical Detail:** A Blind SSRF vulnerability in OpenClaw's cron webhook delivery (`server-cron.ts`) allowed attackers to bypass network policies and hit internal metadata services (AWS IMDS, Kubernetes API) via DNS rebinding or direct IP usage.
*   **Campaign:** Trend Micro identified 39 malicious skills on ClawHub designed to distribute "Atomic MacOS Stealer" by impersonating legitimate CLI tools.
*   **Mitigation:**
    1.  Upgrade OpenClaw to v2026.2.19+.
    2.  Audit installed skills: `openclaw skills list`.
    3.  Verify skill sources before installation.

### AI-Augmented Cloud Intrusions (FortiGate & CrowdStrike Report)
*   **Observation:** Threat actors are using Generative AI to automate the compromise of edge devices. A recent campaign compromised >600 FortiGate devices using AI-generated scripts to adapt to specific firmware versions and evade static detection.
*   **IoC Profile:** Attacks often originate from ephemeral cloud instances (AWS/Azure) rather than traditional botnets, using valid credentials harvested via "Prompt Injection" attacks against internal corporate AI bots.
