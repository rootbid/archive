# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This brief details critical vulnerabilities actively exploited in the wild or with public Proof-of-Concept (PoC) code as of February 27, 2026. High-impact threats targeting cloud infrastructure, remote access systems, and web engines dominate the landscape. Immediate patching and mitigation are required for Chrome/Chromium-based services, Cisco SD-WAN, Ivanti EPMM, and BeyondTrust instances.

## Critical Vulnerabilities & Active Threats

### 1. Google Chrome/Chromium: Zero-Day RCE
*   **CVE:** CVE-2026-2441
*   **Severity:** Critical (Zero-Day)
*   **Type:** Use-After-Free in Blink CSS engine.
*   **Status:** **Active In-the-Wild Exploitation** | PoC Publicly Available.
*   **Impact:** Remote Code Execution (RCE) in the renderer process.
*   **Target:** Desktop users and **headless cloud workloads** (CI/CD pipelines, scraping bots, PDF generators).
*   **Mitigation:** Update Chrome/Chromium to latest version immediately. Restart instances.

### 2. Cisco SD-WAN: Unauthenticated Admin Access
*   **CVE:** CVE-2026-20127
*   **Severity:** Critical (CVSS 10.0)
*   **Type:** Authentication Bypass.
*   **Status:** **Active Exploitation** (Confirmed since 2023).
*   **Impact:** Unauthenticated remote attackers can gain **administrative (root) privileges** on the device.
*   **Target:** Cisco Catalyst SD-WAN Controller/Manager.
*   **Mitigation:** Apply Cisco emergency patches immediately. Restrict management interface access to trusted IPs.

### 3. Ivanti EPMM (MobileIron): Unauthenticated RCE
*   **CVEs:** CVE-2026-1281, CVE-2026-1340
*   **Severity:** Critical (CVSS 9.8)
*   **Type:** Remote Code Execution.
*   **Status:** **Active Exploitation**.
*   **Impact:** Unauthenticated attackers can execute arbitrary commands on the appliance.
*   **Target:** Ivanti Endpoint Manager Mobile (formerly MobileIron).
*   **Mitigation:** Apply hotfixes provided by Ivanti. Isolate appliances from the public internet if possible.

### 4. BeyondTrust Privilege Management: Pre-Auth RCE
*   **CVE:** CVE-2026-1731
*   **Severity:** Critical (CVSS 9.9)
*   **Type:** Pre-authentication Remote Code Execution.
*   **Status:** **Active Exploitation** (Deployment of VShell & SparkRAT).
*   **Impact:** Full system compromise before authentication. Used to deploy persistence tools.
*   **Target:** BeyondTrust Privilege Management for Windows/Mac/Linux (PMW/PMfM/PML).
*   **Mitigation:** Upgrade to patched versions immediately.

### 5. Infrastructure & Libraries (OpenSSL & Node.js Ecosystem)
*   **OpenSSL:** Recent consolidated update (Jan 2026) addressed a **High-severity stack buffer overflow** in CMS AuthEnvelopedData (potential RCE). Ensure OpenSSL is updated to the latest stable release.
*   **Flask-Reuploaded (Python/Web):** CVE-2026-27641 (CVSS 9.8) - Unauthenticated RCE via path traversal.
*   **Node.js/Automation:** **OpenLIT** and **n8n** (CVE-2025-68613) workflow automation tools have recent RCE disclosures. Verify security of internal automation platforms.

## Indicators of Compromise (IoCs)
*   **Traffic:** Anomalous outbound connections from SD-WAN or MDM appliances to unknown IPs.
*   **Files:** Presence of `VShell` or `SparkRAT` binaries on privilege management servers.
*   **Logs:**
    *   *Cisco:* Unexpected admin logins from external IPs.
    *   *Chrome:* Crashes in renderer processes associated with specific CSS parsing.
