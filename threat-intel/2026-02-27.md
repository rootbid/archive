# Sentinel Threat Intelligence: 2026-02-27

## Executive Summary
This report highlights critical vulnerabilities actively impacting global infrastructure as of February 27, 2026. Immediate attention is required for a widely exploited Chrome Zero-Day targeting cloud workloads and critical RCEs in AI engineering platforms and industrial control web interfaces.

## 1. Google Chrome/Chromium Zero-Day RCE (CVE-2026-2441)
**Severity:** Critical | **Status:** Actively Exploited In-the-Wild
**Target:** Cloud Workloads (Headless Chrome), User Workstations

A Use-After-Free vulnerability in the CSS component allows attackers to execute arbitrary code. This is being actively weaponized to compromise CI/CD pipelines and headless browser services used in cloud automation.

*   **IoCs:**
    *   Crashes with "Error code: 11" or "Aw, Snap!".
    *   Unexpected shell processes spawned by chrome/chromium instances.
*   **Mitigation:**
    *   Update Google Chrome/Chromium immediately to the latest stable release.
    *   Restrict headless browser privileges in containerized environments.

## 2. OpenLIT AI Platform Critical RCE (CVE-2026-27941)
**Severity:** Critical (CVSS 9.9) | **Status:** PoC Available
**Target:** AI Engineering & Observability Platforms

A flaw in the OpenLIT platform allows unauthenticated attackers to execute arbitrary code, compromising AI model pipelines and telemetry data. This poses a significant risk to organizations leveraging OpenLIT for LLM observability.

*   **Mitigation:**
    *   Isolate OpenLIT instances behind strict firewalls/VPNs.
    *   Apply vendor patches immediately.

## 3. XWEB Pro Unauthenticated OS Command Injection (CVE-2026-24663)
**Severity:** Critical (CVSS 9.0) | **Status:** Critical
**Target:** Industrial/IoT Web Interfaces

XWEB Pro version 1.12.1 and prior are vulnerable to unauthenticated OS command injection, granting attackers root-level access to the underlying system.

*   **Mitigation:**
    *   Update to version > 1.12.1.
    *   Ensure management interfaces are not exposed to the public internet.
