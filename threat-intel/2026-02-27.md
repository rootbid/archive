# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This report highlights critical vulnerabilities actively exploited in the wild or posing immediate high-risk to infrastructure. The threat landscape for late February 2026 is dominated by **Zero-Day RCEs** in core enterprise tools (Chrome, BeyondTrust, Ivanti) and network infrastructure (Cisco SD-WAN). Immediate patching is required.

## Critical Vulnerabilities & Zero-Days

### 1. Chrome Zero-Day RCE (CVE-2026-2441)
*   **Severity:** **CRITICAL** (Actively Exploited)
*   **Target:** Google Chrome, Chromium-based browsers, Headless Chrome (CI/CD pipelines).
*   **Vector:** Remote Code Execution via malicious HTML/CSS pages.
*   **Status:** **Active Exploitation in the Wild.**
*   **Impact:** Attackers can execute arbitrary code inside the sandbox. Critical risk for users and **automated headless browser tasks** (scrapers, testing suites).
*   **Mitigation:** Update to Chrome 145.0.7632.75+ (Win/Mac) / 144.0.7559.75+ (Linux) immediately.

### 2. BeyondTrust Remote Support RCE (CVE-2026-1731)
*   **Severity:** **CRITICAL (CVSS 9.9)**
*   **Target:** BeyondTrust Remote Support (RS) and Privileged Remote Access (PRA).
*   **Vector:** Pre-authentication Remote Code Execution via crafted HTTP requests.
*   **Status:** **Active Exploitation.** Threat actors are dropping VShell and SparkRAT payloads.
*   **Impact:** Full system compromise of privileged access management appliances.
*   **Mitigation:** Patch RS to 25.3.2+ and PRA to 25.1+.

### 3. Ivanti EPMM Zero-Day RCEs (CVE-2026-1281 & CVE-2026-1340)
*   **Severity:** **CRITICAL**
*   **Target:** Ivanti Endpoint Manager Mobile (formerly MobileIron).
*   **Vector:** Unauthenticated RCE allowing full control over MDM infrastructure.
*   **Status:** **Active Exploitation.**
*   **Impact:** Complete takeover of mobile device management, potentially compromising all managed mobile devices and data.
*   **Mitigation:** Apply immediate vendor patches or isolate appliances from the public internet.

### 4. Cisco Catalyst SD-WAN Auth Bypass (CVE-2026-20127)
*   **Severity:** **CRITICAL**
*   **Target:** Cisco Catalyst SD-WAN Controller/Manager.
*   **Vector:** Authentication Bypass.
*   **Status:** Zero-Day disclosure.
*   **Impact:** Allows unauthorized access to network management controllers, enabling traffic redirection and interception.
*   **Mitigation:** Update to fixed releases immediately.

### 5. Node.js 'basic-ftp' Path Traversal (CVE-2026-27699)
*   **Severity:** **CRITICAL (CVSS 9.1)**
*   **Target:** Node.js applications using `basic-ftp` library.
*   **Vector:** Path Traversal (CWE-22).
*   **Impact:** Unauthorized file access/overwrite on servers using this common library.
*   **Mitigation:** Update `basic-ftp` dependency to latest version.
