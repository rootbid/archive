# Sentinel Threat Intelligence Brief: 2026-02-27

## Executive Summary
This report highlights two critical zero-day vulnerabilities currently under active exploitation in the wild, targeting Google Chrome/Chromium infrastructure and Cisco SD-WAN appliances. Additionally, cumulative critical updates for the Linux kernel and glibc have been released to address arbitrary code execution risks. Immediate patching is mandatory for all affected systems.

## 1. Google Chrome/Chromium: Critical RCE (Zero-Day)
**CVE-2026-2441**
- **Severity**: Critical
- **Type**: Use-After-Free (UAF) in Blink CSS engine.
- **Status**: **Active exploitation in the wild.**
- **Impact**: Remote Code Execution (RCE) via malicious web content. Attackers can execute arbitrary code on the victim's machine. This also affects headless Chromium instances used in cloud automation and CI/CD pipelines.
- **Affected Systems**: Google Chrome, Chromium-based browsers (Edge, Brave, etc.), and Electron apps using vulnerable versions.
- **Mitigation**: Update immediately to Chrome version **145.0.7632.75** or later.
- **IoC**: 22 public Proof-of-Concept (PoC) exploits are available on GitHub. Monitor for unexpected child processes spawned by browser instances.

## 2. Cisco SD-WAN: Authentication Bypass (Zero-Day)
**CVE-2026-20127**
- **Severity**: Critical (CVSS 10.0)
- **Type**: Authentication Bypass.
- **Status**: **Active exploitation reported.** CISA KEV listing.
- **Impact**: Allows an unauthenticated, remote attacker to gain administrative access (root privileges) to the device. This effectively grants full control over the SD-WAN fabric.
- **Affected Systems**: Cisco Catalyst SD-WAN Controller (formerly vSmart) and Cisco Catalyst SD-WAN Manager (formerly vManage).
- **Mitigation**: Upgrade to fixed software releases immediately (e.g., versions 20.9.x patched releases). There are no workarounds.
- **IoC**: Review audit logs for unauthorized administrative logins or configuration changes originating from unknown IP addresses.

## 3. Infrastructure: Critical Linux & Glibc Updates
**Multiple CVEs (Week 7, 2026)**
- **Severity**: High
- **Type**: Arbitrary Code Execution / Privilege Escalation.
- **Status**: Patches released; active risk for unpatched systems.
- **Impact**: Recent updates for Ubuntu and AlmaLinux address critical flaws in the Linux kernel and glibc that could allow local attackers to escalate privileges to root or remote attackers to execute code via specific vectors (e.g., glibc DNS parsing).
- **Mitigation**: Apply all system updates immediately (`apt update && apt upgrade` / `dnf update`). Reboot is required for kernel patches to take effect.
