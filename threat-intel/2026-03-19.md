# Sentinel Threat Intel: 2026-03-19

## Critical Threats Overview

### 1. n8n Pre-Auth RCE (CVE-2025-68613) - Actively Exploited
* **Target:** n8n workflow automation instances (highly used for automated/AI pipelines).
* **Impact:** Critical RCE, added to CISA KEV. Over 24,700 exposed instances.
* **Exploitation:** Actively exploited in the wild.
* **IoCs:** High volumes of requests targeting unauthenticated API endpoints in n8n; execution of unexpected shell processes (e.g., node spawning `/bin/sh` or `/bin/bash`).
* **Mitigation:** Update n8n immediately to the latest patched release. Restrict administrative API access and block public exposure of the n8n web interface using a WAF or VPN.

### 2. Telnetd Unauthenticated Root RCE (CVE-2026-32746)
* **Target:** Systems running the `telnetd` service.
* **Impact:** CVSS 9.8 (Critical). Allows unauthenticated attackers to gain root access.
* **Exploitation:** Pre-auth flaw exposing telnetd via port 23, risking full system takeover.
* **IoCs:** Unexpected connections to port 23; shell commands executed by the telnetd process as root.
* **Mitigation:** Disable `telnetd` immediately. Use secure alternatives such as SSH. Block port 23 at the network firewall.

### 3. Ubuntu systemd Privilege Escalation (CVE-2026-3888)
* **Target:** Ubuntu systems utilizing `snap-confine` and `systemd-tmpfiles`.
* **Impact:** High (Local Root Privilege Escalation).
* **Exploitation:** Allows an unprivileged local attacker to escalate to root via timing exploit interacting with systemd cleanup components.
* **IoCs:** Suspiciously timed executions involving `snap-confine` and `systemd-tmpfiles`; creation of unusual symlinks in temporary directories.
* **Mitigation:** Apply the latest security patches from Ubuntu for the snapd and systemd packages.

### 4. Wing FTP Server Flaws
* **Target:** Wing FTP Server (< 7.4.4).
* **Impact:** Actively exploited server path leaks; related to critical RCE CVE-2025-47812 (CVSS 10.0).
* **Exploitation:** Path traversal and subsequent RCE vectors.
* **Mitigation:** Upgrade to version 7.4.4 or later immediately.
