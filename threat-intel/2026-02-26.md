## 🚨 Critical Infrastructure Alerts

### 1. Google Chrome: CSS-Based Zero-Day (CVE-2026-2441)
**Severity:** CRITICAL | **Status:** Active Exploitation
A zero-day vulnerability in Chrome's CSS handling engine allows malicious websites to execute arbitrary code within the Chrome sandbox via crafted stylesheets.
- **Impact:** Drive-by RCE on updated browsers.
- **Mitigation:** Update to Chrome 146.0.7824.102 immediately. Disable JavaScript/CSS on untrusted sites where feasible.

### 2. OpenSSL: "Dozen-Flaw" Disclosure
**Severity:** HIGH | **Status:** Patch Available
OpenSSL Project has released patches for 12 vulnerabilities, some present for years. While individual severity varies, chained exploitation could lead to RCE or key exfiltration.
- **Impact:** Compromise of TLS termination points and internal CA infrastructure.
- **Mitigation:** Upgrade to OpenSSL 3.4.2 or 3.3.4 immediately.

### 3. Linux Kernel: MediaTek T700 Driver RCE (CVE-2026-23172)
**Severity:** HIGH | **Status:** PoC Likely
A remote code execution vulnerability in the `t7xx` PCIe driver for MediaTek T700 modems.
- **Impact:** RCE on embedded Linux endpoints and IoT gateways using this chipset.
- **Mitigation:** Blacklist the `t7xx` module if unused; apply kernel patches for 6.13+.
